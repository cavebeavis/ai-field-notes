# Notes

This directory is the capture layer for observations, arguments, questions, reactions, and partial ideas about AI.

Notes are allowed to be blunt and incomplete. They should still distinguish what was observed from what is inferred, avoid invented facts, and record enough context to be understandable later.

Use [`../templates/note.md`](../templates/note.md) when starting a substantive note. When a note becomes a coherent, evidence-backed argument, promote it intentionally into [`../papers/`](../papers/) rather than merely changing its tone and pretending it was always settled.
