# Published Work

This directory records material released outside the repository.

For each publication, keep a small Markdown record containing:

- title;
- publication date;
- canonical URL;
- publication venue;
- source document and commit or tag;
- material edits made by the publisher;
- correction, update, or supersession history.

A final snapshot may be stored here when useful, but the canonical external URL and the repository source version must remain clear. Do not duplicate drafts here.
