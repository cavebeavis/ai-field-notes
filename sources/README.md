# Sources

This directory preserves research trails used by notes, papers, and experiments.

Good source material here includes:

- annotated bibliographies;
- links with access dates;
- short summaries of primary sources;
- notes about datasets and benchmark limitations;
- claim-to-source maps;
- conflicting evidence that should not disappear merely because it complicates an argument.

Prefer primary sources for technical claims: research papers, official documentation, standards, source code, filings, and original datasets. Secondary sources may provide context but should not silently become the foundation of a technical claim.

Do not copy complete copyrighted articles or papers into this repository. Store citations, lawful excerpts, summaries, and links instead.
