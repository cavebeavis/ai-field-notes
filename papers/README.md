# Working Papers

This directory contains coherent, evidence-backed long-form work intended for review or eventual publication.

A file belongs here when it has:

- a defined problem and thesis;
- evidence or clearly identified research gaps;
- analysis that exposes assumptions;
- serious counterarguments or alternatives;
- limitations;
- enough structure to be reviewed as a complete argument.

A working paper is not automatically academic research, peer reviewed, or final. Its front matter must state its actual status.

Start from [`../templates/paper.md`](../templates/paper.md). Early observations and undeveloped arguments belong in [`../notes/`](../notes/) instead.
